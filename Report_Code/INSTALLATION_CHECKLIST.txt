================================================================================
TROUBLESHOOTING QUICK LINKS
================================================================================

Issue: Port 5000 already in use
Solution: Use different port or kill process using port

Issue: Module not found
Solution: npm install && npm install --legacy-peer-deps

Issue: CORS Error
Solution: Ensure app.use(cors()) in server.js

Issue: SSAS connection failed
Solution: Check SSAS server is running and accessible

Issue: API returns no data
Solution: Verify backend is running and API endpoints work

Issue: Frontend shows blank page
Solution: Check browser console for errors (F12)

Issue: Data doesn't update on button click
Solution: Check Network tab in DevTools for API calls
