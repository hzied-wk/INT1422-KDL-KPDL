🎓 UNDERSTANDING OLAP OPERATIONS:

OLAP = Online Analytical Processing

It provides 4 basic operations:

1️⃣  DRILL DOWN 🔍
    Look deeper at the data
    Example: "Total" → "By Category" → "By Product"

2️⃣  ROLL UP 🔼
    Combine data into higher level
    Example: "By Product" → "By Category" → "Total"

3️⃣  SLICE & DICE 🎲
    Filter data by conditions
    Example: Show only 2024 data, or North region

4️⃣  PIVOT 🔄
    Rearrange data (swap rows and columns)
    Example: Years as columns, Products as rows

❓ COMMON QUESTIONS:

Q: How do I use real SSAS data instead of mock data?
A: Read SSAS_INTEGRATION.md for detailed instructions

Q: How do I change colors or styling?
A: Edit client/src/App.css

Q: How do I add more cubes?
A: Update config/ssas.config.js and server.js

Q: How do I deploy to production?
A: Read DEPLOYMENT.md

Q: The frontend won't load?
A: Make sure both servers are running:
   Terminal 1: npm start (backend)
   Terminal 2: cd client && npm start (frontend)

Q: I see CORS errors?
A: This shouldn't happen with the current setup,
   but see INSTALLATION_CHECKLIST.txt for troubleshooting

⚡ QUICK COMMANDS:

Start both servers:
  Windows: C:\olap_web\start.bat

  cd C:\olap_web

  cd C:\olap_web\client
