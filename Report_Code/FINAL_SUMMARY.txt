🎯 KEY FEATURES
════════════════════════════════════════════════════════════════════════════════

1️⃣  DRILL DOWN 🔍
    → Xem chi tiết dữ liệu ở mức thấp hơn
    → Mỗi cấp độ hiển thị dữ liệu chi tiết hơn
    → Level counter hiển thị vị trí hiện tại

2️⃣  ROLL UP 🔼
    → Gộp dữ liệu thành mức cao hơn
    → Tính toán tổng hợp tự động
    → Chỉ hoạt động khi level > 0

3️⃣  SLICE & DICE 🎲
    → Lọc dữ liệu theo điều kiện
    → Hỗ trợ multiple filters
    → Hiển thị dữ liệu đã lọc

4️⃣  PIVOT 🔄
    → Sắp xếp lại cấu trúc bảng
    → Hàng ↔ Cột
    → Crosstab format

🔌 API ENDPOINTS
════════════════════════════════════════════════════════════════════════════════

GET  /api/cubes                    → Get available cubes
POST /api/query                    → Query data with MDX
POST /api/dimensions               → Get dimensions
POST /api/operations/drill-down    → Drill down operation
POST /api/operations/roll-up       → Roll up operation
POST /api/operations/slice-dice    → Filter data
POST /api/operations/pivot         → Pivot operation

🐛 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Problem: Port 5000 in use
Solution: netstat -ano | findstr :5000 → taskkill /PID <PID> /F

Problem: CORS Error
Solution: Ensure app.use(cors()); in server.js

Problem: Module not found
Solution: npm install; npm install --legacy-peer-deps

Problem: SSAS connection failed
Solution: Check SSAS is running: telnet localhost 2383

Problem: Frontend shows blank
Solution: Check browser console (F12) for errors

See INSTALLATION_CHECKLIST.txt for more troubleshooting.
