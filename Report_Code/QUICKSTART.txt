📊 TÍNH NĂNG CHÍNH:

1. 🔍 DRILL DOWN (Khoan sâu xuống)
   - Xem chi tiết dữ liệu ở mức thấp hơn
   - Mỗi lần click tăng level
   - Dữ liệu cập nhật tự động

2. 🔼 ROLL UP (Cuộn lên)
   - Gộp dữ liệu lên mức cao hơn
   - Hoạt động khi level > 0
   - Tính toán tổng hợp

3. 🎲 SLICE & DICE (Chiếu chọn)
   - Lọc dữ liệu theo điều kiện
   - Hỗ trợ nhiều filters
   - Hiển thị dữ liệu đã lọc

4. 🔄 PIVOT (Xoay)
   - Sắp xếp lại cấu trúc bảng
   - Hàng ↔ Cột
   - Dạng Crosstab

🐛 TROUBLESHOOTING:

Problem: Port 5000/3000 bị chiếm
Solution:
  netstat -ano | findstr :5000
  taskkill /PID <PID> /F

Problem: CORS Error
Solution:
  Đảm bảo backend đã enable CORS
  Kiểm tra server.js dòng: app.use(cors());

Problem: Module not found
Solution:
  npm install
  npm install --legacy-peer-deps (nếu cần)

Problem: SSAS connection failed
Solution:
  - Kiểm tra SSAS server có chạy
  - Verify host, port trong config
  - Kiểm tra firewall
