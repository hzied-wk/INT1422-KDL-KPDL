================================================================================
                          OLAP OPERATIONS FLOW
================================================================================

Initial Load (GET /api/cubes)
────────────────────────────────
  Frontend → Backend → Return Cube List
  [sales_cube, inventory_cube]

Query Operation (POST /api/query)
──────────────────────────────────
  Frontend → Backend → Build MDX Query → SSAS → Return Data
  Data displayed in Table + Chart

Drill Down (POST /api/operations/drill-down)
──────────────────────────────────────────────
  Level: 0 → 1 → 2 → ...
  Data becomes more detailed at each level

  Example Flow:
  └─ All Products (Level 0)
     └─ Categories (Level 1)
        └─ Products (Level 2)
           └─ Variants (Level 3)

Roll Up (POST /api/operations/roll-up)
────────────────────────────────────────
  Level: 3 → 2 → 1 → 0
  Data becomes more aggregated at each level

  Example Flow:
  └─ Variants (Level 3)
     └─ Products (Level 2)
        └─ Categories (Level 1)
           └─ All Products (Level 0)

Slice & Dice (POST /api/operations/slice-dice)
────────────────────────────────────────────────
  Apply Filters to Data

  Example:
  All Data → Filter (Year=2024, Region=North) → Filtered Data

Pivot (POST /api/operations/pivot)
──────────────────────────────────
  Swap Rows ↔ Columns

  Before Pivot:        After Pivot:
  ┌────┬─────┬─────┐  ┌────┬──────┬────────┐
  │Yr │2023 │2024 │  │Prd │North │South  │
  ├────┼─────┼─────┤  ├────┼──────┼────────┤
  │L   │50k  │60k  │  │L   │15k   │12k    │
  │D   │30k  │35k  │  │D   │8k    │10k    │
  └────┴─────┴─────┘  └────┴──────┴────────┘

================================================================================
                        DATA FLOW DIAGRAM
================================================================================

User Interaction
    ↓
Frontend (React) State Update
    ↓
Call API Endpoint
    ↓
Express Backend Route Handler
    ↓
Apply Business Logic
    ↓
Generate Response
    ↓
Send JSON Response
    ↓
Frontend Receives & Updates State
    ↓
Component Re-renders
    ↓
Update UI (Table, Chart, etc.)
    ↓
User Sees Changes
